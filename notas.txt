# clase robot

atributos
- String nombre: Del robot
- int vida: Puntos de vida. Max 100
- boolean esta vivo: Valor boolean si esta vivo
- int defensa: Puntos de defensa. Max 100
- precisión: Puntos de precisión. Max 100. Estos puntos se usan para determinar si el atacante acerto el tiro.
- arma/arrayList de armas: Una lista con 3 armas que puede portar el robot. Debe elegir cual arma quiere del sorteo antes de empezar. 
Debe elegir cual quiere usar para atacar 

métodos
- aplicar daño: Debe seleccionar que arma de la lista quiere usar y con las estadísticas de esta arma y del atacado se calcula primero la probabilida de acertar el ataque y segundo que tanto daño hizo
presiciónEfectiva = precisión del jugador - retroceso del arma 
ataqueAplicado = ataque - defensa donde el máximo debe ser 100 y el mínimo 1, luego de eso se le agrega una varianza de 0.9 a 1.10 para hacerlo mas impredecible

- recibir daño: Recibe el daño del atacante  
- to string: Devuelve un string de los datos del robot

# clase armas
 
atributos
- String nombre: Del arma
- int daño: Punto de daño con máximo de 100 
- int retroceso: Puntos de retroceso máximo de 100. Esto es para agregar un nerfeo si la arma es muy potente para que sea mas justo el juego


métodos
- to string: muestra los datos del arma
